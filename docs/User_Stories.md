# User Stories

## Student User Stories

### Authentication & Profile Management

- As a student, I want to register on the platform so that I can access its features.
- As a student, I want to log in securely so that I can access my account.
- As a student, I want to update my profile information so that my details remain accurate.
- As a student, I want to upload a profile picture so that other users can identify me.
- As a student, I want to specify my lifestyle and housing preferences so that I can receive personalized recommendations.

---

### Accommodation Search

- As a student, I want to browse available accommodations so that I can find a suitable place to stay.
- As a student, I want to search accommodations by location so that I can find places near my college or preferred area.
- As a student, I want to filter accommodations by budget, amenities, and room type so that I can narrow down my options.
- As a student, I want to view detailed information and images of a property so that I can make informed decisions.
- As a student, I want to compare multiple listings so that I can choose the best accommodation.

---

### Favorites & Recommendations

- As a student, I want to save listings to my favorites so that I can revisit them later.
- As a student, I want to receive personalized accommodation recommendations so that I can discover suitable properties more easily.

---

### Roommate Matching

- As a student, I want to fill out a lifestyle questionnaire so that the system can determine my living preferences.
- As a student, I want to receive a compatibility score with other students so that I can find suitable roommates.
- As a student, I want to view potential roommate profiles so that I can evaluate compatibility before contacting them.

---

### Communication

- As a student, I want to chat with property owners so that I can ask questions about accommodations.
- As a student, I want to chat with potential roommates so that I can discuss living arrangements.

---

### Reviews & Ratings

- As a student, I want to submit reviews and ratings so that I can share my experience with other students.
- As a student, I want to read reviews from other students so that I can make better housing decisions.

---

# Property Owner User Stories

## Authentication & Profile Management

- As a property owner, I want to register on the platform so that I can list my properties.
- As a property owner, I want to log in securely so that I can manage my listings.
- As a property owner, I want to update my profile details so that students can view accurate information.

---

## Listing Management

- As a property owner, I want to create accommodation listings so that students can discover my properties.
- As a property owner, I want to upload property images so that students can view the accommodation.
- As a property owner, I want to edit listing information so that I can keep it updated.
- As a property owner, I want to delete listings that are no longer available.

---

## Communication

- As a property owner, I want to receive messages from interested students so that I can answer their queries.
- As a property owner, I want to communicate with potential tenants before confirming accommodation.

---

## Reviews

- As a property owner, I want to view ratings and reviews on my properties so that I can improve my services.

---

# Administrator User Stories

## User Management

- As an administrator, I want to view all users so that I can monitor platform activity.
- As an administrator, I want to block or remove users who violate platform policies.

---

## Listing Management

- As an administrator, I want to verify accommodation listings so that students can trust the information provided.
- As an administrator, I want to remove fake or inappropriate listings.

---

## Review Moderation

- As an administrator, I want to monitor reviews and reports so that the platform remains trustworthy.

---

## Dashboard & Analytics

- As an administrator, I want to view platform statistics so that I can monitor system usage and growth.
- As an administrator, I want to manage reports and complaints submitted by users.

---

# System User Stories

- As a system, I want to calculate the Living Compatibility Index (LCI) so that students can find compatible roommates.
- As a system, I want to generate personalized recommendations so that students receive relevant accommodation suggestions.
- As a system, I want to notify users about new messages and important updates.
- As a system, I want to maintain secure authentication and authorization so that user data remains protected.
